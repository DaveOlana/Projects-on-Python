# Exam Timetable to Calendar PWA

A Python-based Progressive Web App (PWA) that converts Exam Timetables (PDF/Excel) into personalized Calendar files (.ics).

## Features
- **Upload Timetable:** Supports PDF, CSV, and Excel and Word files.
- **Preview & Edit:** Review extracted exams before exporting.
- **One-Click Export:** Generate `.ics` files for Google Calendar/Outlook.
- **Offline Capable:** Installable as a PWA on desktop and mobile.
